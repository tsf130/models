# 基于机器学习的车牌字符识别系统（HOG + SVM）

## 1 项目简介

本项目实现了一个基于传统机器学习方法的车牌字符识别系统。系统采用 **方向梯度直方图（HOG）特征 + 支持向量机（SVM）分类模型** 的技术路线，对车牌图像进行字符分割、特征提取与分类识别，并提供图形化交互界面，支持用户上传车牌图片并查看识别结果。

该项目适用于机器学习课程设计或相关实验研究，具有良好的可复现性与清晰的工程结构。

---

## 2 运行环境

在运行本项目之前，请确保已正确配置以下运行环境：

- 操作系统：Windows 10 / Windows 11  
- Python 版本：Python 3.8 及以上  

### 主要依赖库

- OpenCV  
- NumPy  
- scikit-learn  
- joblib  
- gradio  

可通过以下命令安装依赖：

``` bash
pip install opencv-python numpy scikit-learn joblib gradio
```

## 4 程序运行流程
### 4.1 模型训练（首次运行必须执行）

在项目根目录下进入 src 目录，运行以下命令：

python svm_train.py


程序将完成以下操作：

读取训练集标注文件 train.txt

对车牌图像进行字符分割

提取字符的 HOG 特征

训练 SVM 字符分类模型

将训练好的模型保存为 models/svm_char.pkl

若 models/svm_char.pkl 文件已存在，可跳过此步骤。

### 4.2 模型测试与评估

运行以下命令对模型性能进行评估：

python svm_eval.py


程序功能包括：

加载训练好的 SVM 模型

使用验证集样本进行测试

统计字符级识别准确率

输出不同字符类别的识别效果

该步骤用于验证模型性能，不影响系统正常使用。

### 4.3 启动图形化界面（系统使用）

运行以下命令启动图形化交互界面：

python ui.py


程序启动后：

浏览器中将自动打开识别界面

用户可上传车牌图片

系统自动完成字符识别

界面中显示最终识别结果

## 5 使用说明与注意事项

在运行 ui.py 之前，请确保已生成 svm_char.pkl 模型文件

建议上传清晰、完整的车牌图像以获得较好识别效果

若更换数据集或修改训练参数，需要重新运行 svm_train.py
